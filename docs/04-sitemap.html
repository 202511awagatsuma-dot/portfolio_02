No. セクション名 セクションの目的 主要コンテンツ

3 ユーザーフロー

メインシナリオ 必須
ペルソナがサイトに訪問してから目的を達成するまでの動線（メインの行動パターン）

広告やSNSからLPに訪問

→ ファーストビューでStikaがどのようなサービスかを把握する

→ 主な機能紹介を見て、何ができるかを理解する

→ 利用後の変化や得られる価値を読み、自分にとってのメリットを感じる

→ 利用イメージや先行体験ユーザーの声を見て、使用するイメージと安心感を持つ

→ 料金・利用開始案内を確認する

→ CTAを押して新規登録へ進む

→ 登録完了後、利用を開始する

サブシナリオ 任意
その他の行動パターン（1〜2パターン）

サブシナリオ案 1

LPに訪問

→ ファーストビューと機能紹介を中心に確認する

→ すぐには登録せず、料金や利用方法を確認する

→ 最終クロージングまで読み進める

→ 最後のCTAから新規登録へ進む


サブシナリオ案 2

LPに訪問

→ 主な機能紹介と利用後の変化を確認する

→ 先行体験ユーザーの声を読み、安心感を持つ

→ 自分にも続けられそうだと感じる

→ 中盤または最後のCTAから新規登録へ進む
